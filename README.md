#### 성능
- 각 페이지 별 메타 태그를 추가함.
- jpg, png 사진 파일들을 avif로 수정함.
- 크기가 정해져 있지 않는 파일들의 각각의 크기(높이,너비)를 지정해 줌.

#### 접근성
- 모든 사용자가 동등하게 웹사이트를 이용할 수 있게 접근성을 고려함.
- tab index를 사용하여 키보드로 이용 가능하게 함.
- aria-label을 사용하여 스크린리더 사용자에게 정확한 정보를 전달하도록 함.
